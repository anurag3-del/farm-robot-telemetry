# Farm Robot Telemetry & Motor Sizing Tools

Projects
1. Motor Sizing Calculator
A GUI-based engineering calculator for sizing motors for farm robots.

**Features:**
- Input: robot weight, slope angle, terrain type, wheel radius, speed
- Output: required torque, RPM, motor power, gear ratio
- Exports a professional PDF report
